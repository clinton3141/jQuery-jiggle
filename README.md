jQuery Jiggle
=============

Plugin to make elements jiggle around on mouseover. It takes two settings: distance, speed and recoilSpeed (defaults are 3px, 150ms and 150ms).

$("#navigation a").jiggle({distance: 2, speed: 100, recoilSpeed: 1000});
